CSS = Cascading Style Sheet

Các cách đưa CSS vào wessite

  1. Cách 1: Inline - viết vào thuộc tính style trong thẻ mở của html
  2. Cách 2: Internal - viết trong cặp thẻ style ở phần head
  3. Cách 3: External - viết vào file .css và nhúng vào tài liệu html thông qua thẻ link:css
